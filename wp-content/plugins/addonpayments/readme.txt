The official AddonPayments Plugin for WordPress accepts electronic transactions to your WordPress with a shortcode.

== Description ==

With the official AddonPayments Plugin for WordPress, you can easily sell your product with a shortcode. You will have all your Orders in your WordPress dashboard. This plugin has the aim of being simple and useful for all online merchants.
AddonPayments is the perfect online payments solution for your company, with all the tools you need to manage your ecommerce business.

= Usage =

[addonpayments id="1" price="25" product="SKU2345"]

== Installation ==

1. Upload `addonpayments` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

Or search for AddonPayments in your Dashboard and install it.
